Unix File & User Management Cheat Sheet

Navigation & Basic File Operations

CommandDescription
pwdPrint working directory
lsList files
ls -laList files with details (including hidden)
cd <dir>Change directory
cd ..Go up one directory
cd ~Go to home directory
cd -Go to previous directory
mkdir <name>Create directory
mkdir -p path/a/b/cCreate nested directories
touch <file>Create empty file
cp <src> <dest>Copy file
cp -r <src> <dest>Copy directory recursively
mv <src> <dest>Move/rename file or directory
rm <file>Remove file
rm -r <dir>Remove directory recursively
rm -rf <dir>Force remove (use with caution!)

File Permissions

CommandDescription
chmod 755 <file>Set permissions (rwxr-xr-x)
chmod +x <file>Add execute permission
chmod -x <file>Remove execute permission
chmod -R 644 dir/Set files to rw-r--r--
chmod -R 755 dir/Set files to rwxr-xr-x
chown user:group <file>Change owner
chown -R user:group <dir>Change owner recursively
chgrp <group> <file>Change group

Permission Codes

r = 4, w = 2, x = 1
7 = rwx, 6 = rw-, 5 = r-x, 4 = r--, 0 = ---

CommandDescription

exitExit current user/shell

Creating & Deleting Users

# Add user
sudo useradd -m <username>        # Create with home dir
sudo useradd -m -s /bin/bash <username>  # With bash shell

# Delete user
sudo userdel <username>            # Remove user
sudo userdel -r <username>         # Remove user + home dir

# Modify user
sudo usermod -l <newname> <oldname>      # Rename user
sudo usermod -aG <group> <user>          # Add to group

CommandDescription

Useful Tips

  • Tab completion: Press Tab to auto-complete commands/paths
  • History: history shows past commands, Ctrl+R searches
  • Cancel command: Ctrl+C
  • Suspend to background: Ctrl+Z, then bg to continue
  • Clear terminal: Ctrl+L or clear
  • Emergency exit: Ctrl+\
